# orb_bug

This small project shows one possible bug computing ORB descriptors using OpenCV 3.0.0

##Platform details:

- **OS**: OSX 10.10.5
- **Compiler**:
Apple LLVM version 7.0.0 (clang-700.0.72)
Target: x86_64-apple-darwin14.5.0
Thread model: posix

##Compilation:

    mkdir build && cd build
    ccmake ..
    make
    ./bin/orb_bug
